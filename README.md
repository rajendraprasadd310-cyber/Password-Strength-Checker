# Password Strength Checker

## Description
A simple Python program that checks whether a password is Weak, Moderate, or Strong based on its length and character types.

## Features
- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects digits
- Detects special characters

## Technologies Used
- Python

## How to Run

```bash
python password_checker.py
```

## Example

Input:
```
Password@123
```

Output:
```
Strong Password
```
